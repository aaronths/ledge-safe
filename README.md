# LedgeSafe

At least 13 people die per year in Australia due to rock fishing. Our mobile app LedgeSafe shows anglers real-time risk and nearby conditions for any location. Using tailored algorithms and AI, LedgeSafe reduces the learning curve for analysing conditions, allowing for a safer rock fishing experience for all.

---

## Run the app with Expo (simulator)

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Start the development server**

   ```bash
   npx expo start
   ```

3. **Open in a simulator**
   - **iOS simulator:** With the dev server running, press `i` in the terminal, or run:
     ```bash
     npx expo start --ios
     ```
   - **Android emulator:** With the dev server running, press `a` in the terminal, or run:
     ```bash
     npx expo start --android
     ```

   Ensure an iOS simulator or Android emulator is already open (e.g. via Xcode → Open Developer Tool → Simulator, or Android Studio → Device Manager). The app will install and launch in the simulator.

## Get started (development)

You can start developing by editing the files inside the **app** directory. This project uses [file-based routing](https://docs.expo.dev/router/introduction).

Other ways to run the app:

- [Development build](https://docs.expo.dev/develop/development-builds/introduction/)
- [Expo Go](https://expo.dev/go) on a physical device (scan the QR code from `npx expo start`)
